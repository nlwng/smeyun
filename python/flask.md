# 安装
```
sudo pip install virtualenv  
sudo apt-get install python-virtualenv

$ mkdir myproject
$ cd myproject
$ virtualenv venv
```
无论何时你想在某个项目上工作，只需要激活相应的环境:  
```
. venv/bin/activate
pip install Flask
```
